# Lingceng

我做一些偏实用的小工具，也记录一些可复用的工作流和想法。

## 最近做的小工具

### [打牌计分器](https://pkscore.netlify.app)

四人打牌记分工具。滚轮输入分数，输入三人后自动计算第四人，支持历史记录与撤销，数据本地保存，离线可用。

- 主入口：[pkscore.netlify.app](https://pkscore.netlify.app)
- 备用入口：[lingceng.github.io/scorecard/](https://lingceng.github.io/scorecard/)

### [孤岛连接 Island Link](https://island-link.netlify.app/)

跨应用的关键词跳转工具。在各种 App 之间建立桥梁，通过关键词快速跳转到目标内容，打通信息孤岛。
