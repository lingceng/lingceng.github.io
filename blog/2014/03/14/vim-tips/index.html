Vim Tips

Mar 14th, 2014

HERE IS MY vimrc

wonderful block insert

in Visual block mode:
|v_b_I|    I    insert the same text in front of all the selected lines
|v_b_A|    A    append the same text after all the selected lines

search word under cursor

* to search current word, n to next, N to previous

arround tag

:help v_a
at  # arround tag
it  # inner tag

use mark

ma # mark a
`a # go to a
`A # go to A between buffers

autocomplete with filesystem

ctrl-x ctrl-f

jump between windows

:help window

# make the only window
CTRL-W o

# split
CTRL-W s
# split vertically
CTRL-W v

# navigate with h j k l
# go to left
CTRL-W h

# resize with < > + -
# add 10 width
CTRL-W 10>

show white space

:set list

# close list
:set nolist

buffer change

CTRL-I
CTRL-O

scroll page

N  CTRL-D   window N lines Downwards (default: 1/2 window)
N  CTRL-U   window N lines Upwards (default: 1/2 window)

set highlight when search

set hlsearch
set nohlsearch
